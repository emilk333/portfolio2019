Todo list.

UI design.   Frontend development.

Todo list is what it sounds like. I lacked a todo list application with easy allround access and no extra functionality. Therefore, I designed and developed my own todo list application with a Angular 6 frontend and a Firebase Firestore as database and backend. Besides from listing my todos, the application also displays some weather data via API calls to Openweathermap.
